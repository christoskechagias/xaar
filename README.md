## Run

### <b> 1. Clone repo </b> <br/>
<pre>
$ git clone https://github.com/kechayias/xaar.git
$ cd xaar
</pre>
  
### <b> 2. Setup Local MongoDB </b> <br/>
- Install it from [here](https://www.mongodb.com/try/download/community) <br /> 
- Set MONGODB_URL=mongodb://localhost/xaar <br /> 

### <b> 3. Backend </b> <br/>
<pre>
$ npm install  
$ npm run dev 
</pre>

### <b> 4. Frontend </b> <br/>
<pre>
$ cd client 
$ npm install
$ npm start 
</pre>
